# Project_7
This is a project to pass the chapter 7 of the course to become a data analyst. 

He creado una aplicación que genera 2 tipos de gráficos estadísticos de un conjunto de datos predeterminado.
La aplicación tiene una interfaz que da la opción de crear uno de los 2 cualesquiera o ambos a la vez al seleccionar los recuadros.
Una vez creados los gráficos la app permite hacer zoom en distintas partes de cada una segun la preferencia.
